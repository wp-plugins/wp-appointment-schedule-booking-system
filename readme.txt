== Installation ==

1) Copy/Upload 'appointgen-scappointment' folder to the '/wp-content/plugins/' directory
2) Activate 'Wordpress Appointment Schedule Booking System' form wp plugin option in admin area
3) Plugin will appear in the menu bar of WP Dashboard
4) create timeslot/venue/service/schedule for booking.
5) please save primary menu if no rooms gallery or booking calendar is showing.
